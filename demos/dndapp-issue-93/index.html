Issue #93 — Live Difficulty in Encounter Card

Encounter card header now shows live-calculated difficulty instead of the stale template value after enemy scaling.

Before

Card header showed encounter.difficulty (e.g. "Medium") from the template. After scaling enemies up or down in the Scale panel, the Scale panel showed the correct new difficulty (e.g. "Deadly") but the card header remained stale.

After

The card header computes difficulty live using calculateDifficulty() from the engine, passing campEnc.enemyOverrides when present. Falls back to template difficulty when no overrides exist. Both displays are always in sync.

Files Changed

  • src/components/campaign/EncounterListEditor.tsx — Added getEffectiveDifficulty() helper; replaced static encounter.difficulty with the live-calculated value.
