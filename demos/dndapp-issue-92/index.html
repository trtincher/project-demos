Issue #92 — Clickable Campaign Card

Campaign list cards are now fully clickable. View and Delete buttons replaced with a clickable card and a trash icon.

Before

Each card had a View button and a Delete text button at the bottom. Required an extra click to navigate.

After

The entire card is a <Link> that navigates to the campaign detail page. The View button is removed. A 🗑 trash icon sits in the top-right corner; clicking it shows a confirmation dialog before deleting. The click event is stopped from bubbling to prevent navigation.

Files Changed

  • src/pages/dm/CampaignListPage.tsx — Card wrapper changed from <div> to <Link>. Delete button → trash icon SVG with e.preventDefault().
