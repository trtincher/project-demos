Tactical Combat Core Engine

Full D&D 5e combat with Phaser 3 grid visualization, AI-driven enemies, action economy, reactions, death saves, and reward distribution.

Battle Setup

Multiple entry points: Arena Mode (roster characters + encounters), Pre-built Scenarios (curated combat setups), Preset Parties (jump in instantly at Level 1/3/5), and Analyze (batch simulation).

Live Combat

Turn-based tactical combat on a 5ft grid with drag-to-move, click-to-attack, and spell targeting. BG3-inspired action bar shows actions, bonus actions, reactions, spell slots, and class features with tooltips.

  • Full action economy: Action, Bonus Action, Reaction, Movement
  • Critical hits, advantage/disadvantage, cover system (+2/+5 AC)
  • 18 conditions with duration tracking and auto-expiry
  • Death saving throws with stabilize/death mechanics
  • AoE spells: Fireball (sphere), Lightning Bolt (line), Cone of Cold
  • Consumables: Healing Potions, Scrolls, Antitoxin

AI Monster System

Hybrid Behavior Tree + Utility AI with role-aware decisions. 6 monster roles (Melee, Brute, Skirmisher, Ranged, Caster, Healer) each with distinct targeting priorities, preferred range, and retreat thresholds. AI decisions consistently run in <500ms.

Reactions System

  • Shield Spell — +5 AC as reaction to being hit (with "would it save you?" preview)
  • Opportunity Attacks — triggered when enemies leave melee range
  • Protection — impose disadvantage on attacks against adjacent allies
  • Counterspell — contest enemy spellcasting with ability check

DM Tools Creator Suite

DM Mode

Toggle DM Mode before combat to manually control all monster actions. Move enemies on the grid, select weapons/spells, pick targets — full tactical control with a purple-themed UI that distinguishes DM controls from player controls.

Encounter Library & Builder

Browse 13 pre-built encounters or create custom ones. Encounter builder with monster search, count selection, difficulty auto-calculator (5e DMG XP thresholds), and reward configuration. Filter by difficulty tier.

Campaign System

Link a party to an ordered sequence of encounters. Track progress (pending/completed/skipped), auto-mark encounters on victory. Full CRUD with IndexedDB persistence.

Encounter Analysis Engine

Batch simulation runs N combats (10/50/100) with AI controlling both sides. Reports win rate, average rounds, casualty rates, damage distribution, and per-unit breakdowns. Automated difficulty assessment from statistical outcomes.

Homebrew & Data Extensible

Monster Builder

Full homebrew monster creation: ability scores, CR (auto-calculates proficiency + XP), hit points, AC, speed, damage resistances/immunities, weapons, AI role, gold/loot drops. Custom monsters are immediately usable in encounters.

Import/Export & Settings

JSON import/export for characters. Offline-first architecture with IndexedDB + optional Supabase cloud sync. No external API dependencies for gameplay.
